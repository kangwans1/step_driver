# step_driver
- stm32f103 ( blue pill )
-   axis x ( motor 0 )
-     PA0 PULSE pin
-     PA1 DIR pin
-     PB12 LIMIT0 ( active low )
-     PB13 LIMIT1 ( active low )
-   axis y (motor 1)
-     PA6 PULSE pin
-     PA5 DIR pin
-     PB14 LIMIT2 ( active low )
-     PB15 LIMIT3 ( active low )
